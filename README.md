
An authentic Snake Game built using Python and Tkinter GUI.
Control the snake with arrow keys, eat food, and try not to collide with yourself .

Gameplay Features  
->Food appears at random positions  
-> Score increases as the snake grows  
->Collision detection for walls and self  
->Smooth controls and real-time updates
